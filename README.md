# Product Management CRUD Application

## Overview
Spring Bootを用いた商品管理CRUDアプリケーションです。
要件整理から設計、実装、テスト、設計書作成までを
実務を想定して一通り実施しました。

## Features
- 商品一覧表示
- 商品登録
- 商品編集・更新
- 商品削除
- バリデーション処理

## Tech Stack
- Java
- Spring Boot
- Thymeleaf
- JPA
- MariaDB
- Validation

## Notes
HTMLフォームの制約を踏まえ、更新処理はPOSTで実装しています。
設計・テスト観点は別途資料として整理しています。
